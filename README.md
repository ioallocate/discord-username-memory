# discord-username-memory
> Get's a persons Discord Username from memory

# How to get "Discord Username Offset"
> Open Cheat Engine

> Search for your username

> Click around discord, wait until you notice theres a couple addresses that always have your username

> Generate a pointer map

> Look for the smallest chain ( not required but i dont like doing a shit ton for something simple )
https://imgur.com/a/xqHuIjo

> Replace offset in source.cpp with that including the chain

# What could be done using this?
> Token Grabber's

> Message Loggers

> Anything that the user see's you can access.
